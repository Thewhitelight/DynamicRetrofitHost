# DynamicRetrofitHost
dynamic swtich host,利用Interceptor动态切换host,方便app内切换不同环境

当有测试、mock、预发、生产环境时,对应打Flavors生成不同的apk,感觉太过麻烦,所以在app内动态切换,更加使用方便.
